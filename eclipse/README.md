# eclipse設定関連
## ワーキングセット
- プロジェクトをグルーピングする
- ワーキングセット単位に検索などを実行できる
- [project explorer] -> [select working sets] -> [New...] -> [working set name] -> [add]

## ブラウザ設定
- 外部ブラウザを起動する
- [環境設定] -> [General] -> [Web Browser] -> [Use external web browser]

## インデント文字の変更
- タブ文字からスペース文字のみに変更
- [環境設定] -> [JavaScript] -> [Code Style] -> [Edit...] -> [Tab policy] -> [Spaces only]
